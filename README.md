## 🛠 Tecnologias e Ferramentas

- **Linguagem:** Python, R
- **Banco de Dados:** MySQL 
- **Bibliotecas Comuns:** Pandas, NumPy, Scikit-Learn, Matplotlib
- **Ambiente:** VS Code

## 📝 Notas de Organização

- Nem todas as disciplinas contêm implementações de código. Disciplinas focadas em fundamentos teóricos (como Matemática Computacional ou Modelagem de Dados inicial) podem conter apenas resumos em Markdown (`.md`) ou arquivos de diagramas na pasta `/docs`.
- Cada diretório de disciplina pode conter seu próprio `README.md` com os detalhes específicos dos exercícios e projetos desenvolvidos nela.
